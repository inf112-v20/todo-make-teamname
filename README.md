## How to run
*   Clone the repo in terminal: "git clone https://github.com/inf112-v20/todo-make-teamname.git" or download from zip.
*   Open IntelliJ and press _Open_ and choose the project folder.
*   Navigate to the _Main_ file and choose _Run_ to start the game.

## Known bugs
Currently throws "WARNING: An illegal reflective access operation has occurred", 
when the java version used is >8. This has no effect on function or performance, and is just a warning.
